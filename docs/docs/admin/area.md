# 地区
## 地区列表

```html
GET - /area/list
```

| 参数    | 是否必填 | 说明                                       |
| ----- | ---- | ---------------------------------------- |
| pid   | 否    | 父级id                                     |
| level | 否    | 层级，规则：设置显示下级行政区级数（行政区级别包括：省/直辖市、市、区/县3个级别）可选值：0、1、2  0：返回省/直辖市；1：返回省/直辖市、市；2：返回省/直辖市、市、区/县 |

> 开发这个请注意，数据源从高德去抓取，http://lbs.amap.com/api/webservice/guide/api/district  抓完和我说下 我们看看数据 是否用这个
